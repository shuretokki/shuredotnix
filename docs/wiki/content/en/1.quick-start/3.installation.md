# Installation

## Prerequisites

Before installing, ensure you have:

- NixOS installed (or any Linux with Nix)
- Git installed
- Basic familiarity with Nix/NixOS

## Step 1: Clone the Repository

```bash
git clone https://github.com/shuretokki/dotnix.git
cd dotnix
```

## Step 2: Customize Identity

Edit `identity.nix` with your details:

```nix [identity.nix]
{
  username = "your-username";
  locale = "en_US.UTF-8";
  timezone = "Asia/Jakarta"; # Your Timezone
  theme = "sh";              # or "default"
}
```

## Step 3: Hardware Configuration

Copy your existing hardware configuration to the desktop host:

```bash
cp /etc/nixos/hardware-configuration.nix hosts/desktop/
```

Or generate a new one:

```bash
nixos-generate-config --show-hardware-config > hosts/desktop/hardware-configuration.nix
```

## Step 4: Build and Switch

We use `nh` helper for faster builds and better output:

```bash
nh os switch .#desktop
```

or standard NixOS command:

```bash
sudo nixos-rebuild switch --flake .#desktop
```

::callout{icon="i-lucide-info" color="blue"}
First build may take 30-60 minutes as it downloads and compiles packages.
::

## Binary Caches

The flake includes binary caches for faster builds:

- `shuredotnix.cachix.org` - Pre-built packages
- `hyprland.cachix.org` - Hyprland binaries
- `vicinae.cachix.org` - Vicinae binaries

These are automatically configured in the flake.
