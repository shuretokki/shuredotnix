# Theming (Stylix)

dotnix uses [Stylix](https://github.com/danth/stylix) to enforce a unified look across the entire OS.

## How It Works

One Base16 color scheme + one matching wallpaper = consistent system.

Stylix configures:
- **GTK / QT**: Applications look uniform.
- **Terminals**: Alacritty/Kitty use the scheme colors.
- **Hyprland**: Border colors match the theme.
- **Editors**: VS Code and Neovim themes.

## Changing Themes

Edit `identity.nix` in the root:

```nix
{
  theme = "sh";  # Corresponds to library/display/themes/sh/
}
```

## Creating a Theme

Themes live in `library/display/themes/`. To create a new one:

1. Create a directory (e.g., `library/display/themes/mytheme/`).
2. Add a `default.nix` that defines:
    - `image`: Path to wallpaper.
    - `base16Scheme`: Path to yaml scheme.
    - `stylix.cursor`: Cursor theme settings.
3. Set `theme = "mytheme"` in `identity.nix`.
