# Zen Browser

This document covers the declarative configuration of Zen Browser.

## Policies

| Policy | Value | Rationale |
|--------|-------|-----------|
| `DisableTelemetry` | `true` | Prevents Mozilla/Zen from collecting usage metrics. |
| `DisableFirefoxStudies` | `true` | Blocks A/B testing and "Heartbeat" experiments. |
| `DisablePocket` | `true` | Removes the non-free, proprietary "Read Later" service. |
| `SanitizeOnShutdown` | `true` | Clears cookies/cache on browser exit for privacy. |
| `HttpsOnlyMode` | `"force_enabled"` | Upgrades all HTTP connections to HTTPS automatically. |
| `EnableTrackingProtection` | `Locked: true` | Forces Enhanced Tracking Protection to stay on. |

## Privacy Settings

| Preference | Value | Rationale |
|------------|-------|-----------|
| `privacy.resistFingerprinting` | `true` | Spoofs timezones, fonts, and window sizes to stop tracking. |
| `media.peerconnection.enabled` | `false` | Disables WebRTC to prevent local/real IP leaks (VPN bypass). |
| `webgl.disabled` | `true` | Blocks hardware-based fingerprinting via GPU details. |
| `geo.enabled` | `false` | Completely disables the Geolocation API. |
| `dom.battery.enabled` | `false` | Prevents battery status from being used to track users. |
| `network.trr.mode` | `2` | Uses DNS over HTTPS (DoH) with native fallback. |
| `network.trr.uri` | `cloudflare` | High-performance, privacy-focused DNS resolver. |

## Smaller Compact Mode

**Source**: [n7itro/Smaller-Compact-Mode](https://zen-browser.app/mods/5941aefd-67b0-453d-9b62-9071a31cbb0d/)

This mod reduces the sidebar height in compact mode and limits the expanded sidebar width. It is applied via `userChrome.css` and requires `toolkit.legacyUserProfileCustomizations.stylesheets = true`.

## Extensions

- **uBlock Origin**: The gold standard for content filtering.
- **Bitwarden**: Open-source password management.
- **Dark Reader**: Generates dark mode for every website.
- **SponsorBlock**: Skips YouTube sponsor segments.
- **Auto Tab Discard**: Automatically suspends idle tabs to save RAM.
- **Scroll Anywhere**: Scroll the page with middle-mouse button anywhere.
- **ClearURLs**: Strips tracking parameters (like `?utm_source`) from URLs.
- **Privacy Badger**: EFF-made tracker blocker that learns over time.

## Configuration Structure

The Zen configuration is located in `library/home/bundlers/desktop/browser/`.
- `default.nix`: Main declarative configuration.
- `icons/`: Local SVG assets for custom Spaces.

## Zen Spaces and Containers

| Type | Custom Icon | Notes |
|------|-------------|-------|
| **Spaces** | Yes | Uses local SVGs via Nix paths (e.g., `./icons/code.svg`). |
| **Containers** | No | Restricted to standard Firefox icons (`fingerprint`, `briefcase`, etc.). |

::callout{icon="i-lucide-info" color="blue"}
**Zen Mods**: Most community mods are CSS-based and should be added to the `userChrome` field in `browser.nix`.
::
