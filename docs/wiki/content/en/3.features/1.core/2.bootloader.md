# Limine Bootloader

This guide explains how to use Limine as your bootloader instead of GRUB.

## What You Get

- **Modern**: Clean, fast bootloader with minimal overhead
- **Secure Boot ready**: Can be configured for Secure Boot
- **Dual Boot**: Supports Windows chainloading

## Quick Start

Limine is enabled by default. After your first `nh os boot`, select "NixOS" or "Linux Boot Manager" in your BIOS boot menu.

## Configuration

The bootloader configuration lives in `library/core/boot.nix`.

### Key Settings

```nix
boot.loader.limine = {
  enable = true;
  maxGenerations = 10;  # Limit boot menu entries
  enableEditor = false; # Disable config editor (security)
};
```

### Dual Boot (Windows)

Windows is configured to chainload automatically:

```nix
extraEntries = ''
  /Windows
    protocol: efi_chainload
    path: boot():/EFI/Microsoft/Boot/bootmgfw.efi
'';
```

## Switching from GRUB

If you previously used GRUB:

1. Update your config with Limine enabled
2. Run `nh os boot`
3. Reboot and enter BIOS (F2/F12/DEL)
4. Change boot order to prioritize "NixOS" or "Linux Boot Manager"
5. Optionally remove old GRUB files: `sudo rm -rf /boot/EFI/GRUB`

## Secure Boot

To enable Secure Boot with Limine:

1. Generate keys: `sbctl create-keys`
2. Enter BIOS, reset to "Setup Mode"
3. Enroll keys: `sbctl enroll-keys -m -f`
4. Enable in config: `secureBoot.enable = true`

## Troubleshooting

### Still booting GRUB

Your BIOS is using the old boot entry. Change boot order in BIOS settings.

### Boot menu not showing

Limine boots directly to NixOS by default if only one OS is detected. Hold a key during boot to show the menu.
