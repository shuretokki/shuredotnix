# CON-003: Git Workflow

**Last Updated:** 2026-01-05

## Branch Naming
| Prefix | Format | Use Case |
|--------|--------|----------|
| `feature/` | `feature/T-{id}-{desc}` | New features |
| `bugfix/` | `bugfix/T-{id}-{desc}` | Bug fixes |
| `hotfix/` | `hotfix/T-{id}-{desc}` | Urgent production fixes |
| `release/` | `release/v{version}` | Release branches |
| `docs/` | `docs/T-{id}-{desc}` | Documentation only |

## Pre-Commit
Uses pre-commit framework (pre-commit.com):
- `nix flake check` (5min timeout)
- `flake-checker` (Nixpkgs freshness)
- Auto-installs via `nix develop`

## Bisect for Debugging
```bash
git bisect start && git bisect bad HEAD && git bisect good <tag>
git bisect run nix flake check
git bisect reset  # Always cleanup
```
